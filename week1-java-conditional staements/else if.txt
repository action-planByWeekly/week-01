1. break keyword

ðŸ‘‰ Purpose:

break stops a loop immediately.

When Java sees break, it jumps out of the loop and continues the program after the loop.

Works in:

for loop
while loop
do-while loop
switch statement
-------------------------------------------------------------------------------------------------
ðŸ’¡ Example 1: Stop a loop when found a number
for(int i = 1; i <= 10; i++) {
    if(i == 5) {
        break;   // stop loop completely
    }
    System.out.println(i);
}

Output:
1
2
3
4


The loop stops at 5 and exits

Why use break?

To stop searching when something is found
To exit early
To terminate a loop based on condition
To exit switch-case

2. continue keyword

ðŸ‘‰ Purpose:

continue skips the current iteration of the loop

The loop continues to the next iteration.
-------------------------------------------------------------------------------------------------
ðŸ’¡ Example 1: Skip number 5

    if(i == 5) {

    }

2
3
4
6
7
8
9
10


The loop does NOT stop, only skips printing 5.

-------------------------------------------------------------------------------------------------
Why use continue?

To skip unwanted values
To avoid executing some part of code for a specific condition
To filter items inside loops
