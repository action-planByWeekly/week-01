1.What Is switch ?

A switch statement is a special decision-making structure that replaces multiple if-else-if conditions that check the same variable.

In simple words:

switch checks one variable, and depending on its value, it chooses a matching case and runs that block of code.

Common use-cases:

Checking menu selections

Checking days of the week
Checking user roles
Checking grades
Checking commands
Reading single-character inputs

--------------------------------------------------------------------------------------------------------------------------------------
2. Why does switch exist when we have if?

Because sometimes you need to check one variable against many possible fixed values.

Example:

if(day == 1) { ... }
else if(day == 2) { ... }
else if(day == 3) { ... }
else if(day == 4) { ... }


This becomes messy, long, and hard to read.

switch solves this nicely:

switch(day) {
    case 1: ...
    case 2: ...
    case 3: ...
}

Much cleaner, easier to understand, and less error-prone.

5.  Syntax

    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
}
