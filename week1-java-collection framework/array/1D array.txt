1Ô∏è‚É£ What is a 1D Array?

A 1D array is a collection of elements of the same type stored in contiguous memory locations.

Elements are accessed using an index, starting from 0.

Example declaration:

int[] numbers = new int[5]; // array of 5 integers, all initialized to 0


Or:

int[] numbers = {1, 2, 3, 4, 5}; // array with initial values
-----------------------------------------------------------------------------------------
2Ô∏è‚É£ Declaration and Initialization

Declaration:

int[] arr;     // preferred style
int arr2[];    // also allowed but less common


Initialization:

arr = new int[5]; // allocates memory for 5 integers


Declaration + Initialization together:

int[] arr = new int[5];
int[] arr2 = {1, 2, 3, 4, 5};


‚úÖ Must know:

Default values:

int ‚Üí 0

double ‚Üí 0.0

Object ‚Üí null

3Ô∏è‚É£ Accessing Elements

Use index to get or set values:

int[] arr = {10, 20, 30, 40, 50};
System.out.println(arr[2]); // 30
arr[2] = 100;              // changes 3rd element to 100


OutOfBounds Exception: Accessing arr[5] in above array will throw ArrayIndexOutOfBoundsException.

5Ô∏è‚É£ Iterating Through Arrays

For loop:

    System.out.println(arr[i]);
}


Enhanced for loop:

    System.out.println(num);
}


‚úÖ Must know: Use enhanced for loop when you don‚Äôt need the index.

6Ô∏è‚É£ Common Operations
a) Sum of all elements
int sum = 0;
for(int num : arr) sum += num;
System.out.println("Sum: " + sum);

b) Find maximum & minimum
int max = arr[0], min = arr[0];
for(int num : arr){
    if(num > max) max = num;
    if(num < min) min = num;
}
System.out.println("Max: " + max + ", Min: " + min);

c) Searching

Linear search (basic):

int target = 30;
boolean found = false;
for(int num : arr){
    if(num == target) {
        found = true;
        break;
    }
}
System.out.println(found ? "Found" : "Not Found");
-----------------------------------------------------------------------------------------
7Ô∏è‚É£ Copying Arrays

Using Arrays.copyOf():

9Ô∏è‚É£ Reversing Arrays

for(int i = 0; i < arr.length/2; i++){
    int temp = arr[i];
    arr[i] = arr[arr.length-1-i];
    arr[arr.length-1-i] = temp;

üîü Multi-use Tricks

        sorted = false;

    System.out.println(arr[i]);
}
