It stores unique elements (like HashSet).
Maintains insertion order â€” the order in which elements are added is preserved.
Slightly slower than HashSet due to linked list maintenance.

ğŸ”µ 3. Internal Structure

LinkedHashSet is basically:

HashMap + Doubly Linked List

Each element is stored as a key in an internal HashMap (like HashSet)
The HashMap entry is extended to store pointers to previous and next entries
This doubly linked list preserves insertion order
