1. What is a LinkedList?

A LinkedList is a data structure made of nodes.

Each node contains two things:

      1.Data (the actual value)
      2.Links (pointers)
	one link pointing to the next node
	one link pointing to the previous node
	(because Java LinkedList is doubly-linked)

So a LinkedList looks like this:

2. Why LinkedList Exists? (Purpose)

Because ArrayList is slow when:

	inserting elements in the middle
	removing elements in the middle

Why?

Because ArrayList must shift elements, which takes O(n) time.

LinkedList solves this by using nodes instead of array positions.

10. ArrayList vs LinkedList

| Feature          | ArrayList        | LinkedList              |
| ---------------- | ---------------- | ----------------------- |
| Structure        | Array            | Nodes                   |
| Access by index  | Very fast (O(1)) | Very slow (O(n))        |
| Insert at end    | Fast             | Fast                    |
| Insert in middle | Slow (shifting)  | Fast (pointer change)   |
| Delete in middle | Slow (shifting)  | Fast                    |
| Memory usage     | Less             | More (nodes + pointers) |
